## Transaction
desc of service

<a name="pactus.transaction.get_transaction"></a>
### pactus.transaction.get_transaction

Creates a new session.

##### Request

```json
{
  "jsonrpc": "2.0",
  "id": "1234567890",
  "method": "Session.Login",
  "params": {
    "name": "admin",
    "password": "123456"
  }
}
```

#### Response

```json
{
  "jsonrpc": "2.0",
  "id": "1234567890",
  "result": {
    "session_token": "123456890",
    "validity": 3600
  }
}
```
